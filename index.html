Welcome to GenGen

GenGen is a fast, flexible static site generator built with Dart. It keeps Jekyll's familiar content model while shipping a modern toolchain.

Quick Start
  1. Install dart pub global activate gengen
  2. Create gengen new my-site
  3. Serve gengen serve
