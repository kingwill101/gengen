This demo shows how to use Tailwind CSS with GenGen

Easy Setup

TailwindPlugin is automatically included in GenGen. Just add your Tailwind CSS file and executable.

Auto Compilation

CSS is compiled automatically after rendering, ensuring your styles are ready for production.

Customizable

Configure input, output paths and executable location to fit your project structure.

Getting Started

  1. Download the Tailwind CSS standalone CLI from tailwindcss.com
  2. Place the tailwindcss executable in your project root
  3. Create assets/css/tailwind.css with your Tailwind directives
  4. Build your site - TailwindPlugin will automatically compile your CSS!
