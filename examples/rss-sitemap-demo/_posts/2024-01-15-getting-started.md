# Getting Started with GenGen

Welcome to GenGen! This is your first step into the world of modern static site generation.

## What is GenGen?

GenGen is a powerful, flexible static site generator built with Dart. It provides:

- **Fast builds** with parallel processing
- **Plugin system** for extensibility  
- **Liquid templating** for dynamic content
- **Markdown support** with custom extensions
- **Asset processing** including Sass and Tailwind CSS

## Quick Start

1. Install GenGen
2. Create a new site
3. Add your content
4. Build and deploy
