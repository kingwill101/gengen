ğŸ”§ How It Works

Both plugins are enabled by default and require no configuration:

  1. RssPlugin scans your posts and generates /feed.xml
  2. SitemapPlugin scans all content and generates /sitemap.xml
  3. Both files are created in your output directory after rendering

âš™ï¸ Customization

You can customize these plugins when using GenGen as a library:

// Custom RSS settings
RssPlugin(
  outputPath: 'my-feed.xml',
  maxPosts: 50,
)

// Custom sitemap settings
SitemapPlugin(
  outputPath: 'my-sitemap.xml',
)

For pages, you can also customize sitemap settings in the front matter:

---
title: My Page
sitemap_priority: 0.8
sitemap_changefreq: weekly
---
